# Punto RAW Assets

Static assets for Punto RAW podcast announcements (portraits, badges, etc.)

## Assets

- `/assets/rj.jpg` — RJ portrait for email announcements
- `/assets/rafa.jpg` — Rafa portrait (reference)
- `/assets/spotify-badge.png` — Spotify CTA badge

## Usage

Reference images in emails:
```html

```
